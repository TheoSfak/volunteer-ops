1. Ξεκινώντας — Σύνδεση & Εγγραφή

1.1 Εγγραφή νέου εθελοντή

Αν δεν έχετε λογαριασμό, μπορείτε να εγγραφείτε ακολουθώντας αυτά τα βήματα:

  1. Ανοίξτε τη σελίδα σύνδεσης και πατήστε Εγγραφή
  2. Συμπληρώστε:
    • Ονοματεπώνυμο: Το πλήρες όνομά σας
    • Email: Μια έγκυρη διεύθυνση email
    • Κωδικός: Τουλάχιστον 6 χαρακτήρες
    • Τηλέφωνο: Αριθμός επικοινωνίας
  3. Πατήστε Εγγραφή
  4. Ελέγξτε το email σας για τον σύνδεσμο επιβεβαίωσης (αν απαιτείται)
Σημείωση Η εγγραφή μπορεί να μην είναι διαθέσιμη αν ο οργανισμός έχει απενεργοποιήσει τη λειτουργία αυτο-εγγραφής. Σε αυτή την περίπτωση, ο λογαριασμός σας δημιουργείται από τον διαχειριστή.

1.2 Σύνδεση

  1. Ανοίξτε τη σελίδα σύνδεσης
  2. Εισάγετε το Email και τον Κωδικό σας
  3. Πατήστε Σύνδεση

1.3 Ξέχασα τον κωδικό μου

  1. Στη σελίδα σύνδεσης, πατήστε Ξέχασα τον κωδικό
  2. Εισάγετε το email του λογαριασμού σας
  3. Θα λάβετε email με σύνδεσμο επαναφοράς κωδικού
  4. Ακολουθήστε τον σύνδεσμο και ορίστε νέο κωδικό

1.4 Πλοήγηση στην εφαρμογή

Μετά τη σύνδεση, η εφαρμογή αποτελείται από:

  • Πλαϊνή μπάρα (αριστερά): Μενού πλοήγησης με όλες τις ενότητες
  • Πάνω μπάρα (δεξιά): Ειδοποιήσεις, Προφίλ, Αποσύνδεση
  • Κεντρικό περιεχόμενο: Η σελίδα που βρίσκεστε

2. Πίνακας Ελέγχου

2.1 Προσωπικά στατιστικά

Ο Πίνακας Ελέγχου είναι η πρώτη σελίδα μετά τη σύνδεση. Εμφανίζει τα προσωπικά σας στατιστικά:

  • Βάρδιες: Πόσες βάρδιες έχετε ολοκληρώσει
  • Ώρες: Συνολικές ώρες εθελοντισμού
  • Πόντοι: Τρέχον σύνολο πόντων
  • Επιτεύγματα: Πόσα έχετε κερδίσει

2.2 Επερχόμενες βάρδιες

Στο κάτω μέρος εμφανίζονται οι εγκεκριμένες βάρδιες σας που πρόκειται να πραγματοποιηθούν, με ημερομηνία, ώρα και τοποθεσία.

2.3 Πρόσφατη δραστηριότητα

Δείτε τις τελευταίες σας ενέργειες: αιτήσεις, εγκρίσεις, πόντοι που κερδίσατε.

3. Το Προφίλ μου

Μεταβείτε στο προφίλ σας κάνοντας κλικ στο avatar σας πάνω δεξιά → Το Προφίλ μου.

3.1 Προσωπικά στοιχεία

Στο προφίλ σας μπορείτε να ενημερώσετε:

  • Ονοματεπώνυμο
  • Τηλέφωνο
  • Βιογραφικό σημείωμα
  • Διεύθυνση
  • Επαφή έκτακτης ανάγκης
  • Διαθεσιμότητα: Σημειώστε πότε είστε διαθέσιμοι

3.2 Φωτογραφία προφίλ

  1. Στο δεξί μέρος του προφίλ, πατήστε Αλλαγή φωτογραφίας
  2. Επιλέξτε αρχείο (JPG, PNG, GIF ή WebP, μέγεθος έως 5MB)
  3. Η φωτογραφία περικόπτεται αυτόματα σε 250×250 pixels
  4. Πατήστε Ανέβασμα για αποθήκευση

3.3 Δεξιότητες

Προσθέστε τις δεξιότητές σας (π.χ. Πρώτες Βοήθειες, Οδήγηση, Αγγλικά) για να βοηθήσετε τους διαχειριστές να σας αναθέσουν κατάλληλες αποστολές.

3.4 Αλλαγή κωδικού

  1. Στο προφίλ, βρείτε την κάρτα Αλλαγή Κωδικού
  2. Εισάγετε τον τρέχοντα κωδικό σας
  3. Εισάγετε τον νέο κωδικό (τουλάχιστον 6 χαρακτήρες)
  4. Επιβεβαιώστε τον νέο κωδικό
  5. Πατήστε Αλλαγή Κωδικού

3.5 Πρόοδος αποστολών

Στο πάνω μέρος του προφίλ εμφανίζεται η πρόοδός σας στον ετήσιο στόχο αποστολών (10 αποστολές/έτος) ως μπάρα ποσοστού.

Συμβουλή Κρατήστε το προφίλ σας ενημερωμένο! Οι διαχειριστές χρησιμοποιούν τις δεξιότητες και τη διαθεσιμότητά σας για τον σχεδιασμό αποστολών.

4. Αποστολές & Βάρδιες

4.1 Εύρεση αποστολών

Μεταβείτε στο Ενεργές Αποστολές από το μενού. Εδώ εμφανίζονται όλες οι αποστολές με κατάσταση «Ανοιχτή» που δέχονται εθελοντές.

4.2 Φιλτράρισμα

Χρησιμοποιήστε τα φίλτρα για να βρείτε αποστολές:

  • Τμήμα: Εμφάνιση αποστολών συγκεκριμένου τμήματος
  • Τύπος: Εκπαιδευτική, Επιχειρησιακή κ.λπ.
  • Αναζήτηση: Πληκτρολογήστε λέξεις-κλειδιά στον τίτλο

4.3 Προβολή αποστολής

Κάντε κλικ σε μια αποστολή για να δείτε:

  • Περιγραφή: Τι αφορά η αποστολή
  • Ημ/νίες: Πότε αρχίζει και τελειώνει
  • Τοποθεσία: Πού θα πραγματοποιηθεί
  • Βάρδιες: Λίστα διαθέσιμων βαρδιών με ώρες και θέσεις

4.4 Κατανόηση κατάστασης αποστολής

ΚατάστασηΤι σημαίνει για εσάς
ΑνοιχτήΜπορείτε να αιτηθείτε συμμετοχή
ΚλειστήΔεν δέχεται νέες αιτήσεις
ΟλοκληρωμένηΈχει ολοκληρωθεί
ΑκυρωμένηΈχει ακυρωθεί — θα λάβετε ειδοποίηση αν είχατε εγκεκριμένη συμμετοχή

5. Αίτηση Συμμετοχής

5.1 Πώς αιτούμαι συμμετοχή

  1. Ανοίξτε μια Ανοιχτή αποστολή
  2. Δείτε τις διαθέσιμες Βάρδιες — ελέγξτε ώρες και κενές θέσεις
  3. Πατήστε Αίτηση Συμμετοχής στη βάρδια που θέλετε
  4. (Προαιρετικά) Προσθέστε σημειώσεις στην αίτησή σας
  5. Πατήστε Υποβολή

5.2 Μετά την υποβολή

Η αίτησή σας μπαίνει σε κατάσταση Εκκρεμεί. Ένας διαχειριστής θα τη δει και θα αποφασίσει:

ΚατάστασηΤι σημαίνειΤι κάνετε
Εκκρεμεί Αναμένει απόφαση Περιμένετε — μπορείτε να ακυρώσετε αν αλλάξετε γνώμη
Εγκεκριμένη Η αίτηση εγκρίθηκε! Παρουσιαστείτε στη βάρδια σύμφωνα με τις οδηγίες
Απορρίφθηκε Η αίτηση δεν εγκρίθηκε Ελέγξτε τον λόγο απόρριψης — μπορείτε να αιτηθείτε σε άλλη βάρδια
Ειδοποίηση Θα λάβετε email και ειδοποίηση στην εφαρμογή κάθε φορά που αλλάζει η κατάσταση της αίτησής σας.

5.3 Ακύρωση αίτησης

Μπορείτε να ακυρώσετε μια αίτηση μόνο όταν βρίσκεται σε κατάσταση «Εκκρεμεί»:

  1. Μεταβείτε σε Οι Συμμετοχές μου
  2. Βρείτε την αίτηση
  3. Πατήστε Ακύρωση
Προσοχή Αν η αίτησή σας είναι ήδη εγκεκριμένη, δεν μπορείτε να την ακυρώσετε μόνοι σας. Επικοινωνήστε με τον διαχειριστή.

6. Οι Συμμετοχές μου

6.1 Λίστα συμμετοχών

Στη σελίδα Οι Συμμετοχές μου (από το μενού ή το dropdown πάνω δεξιά) βλέπετε όλες τις αιτήσεις σας:

  • Τίτλος αποστολής και βάρδιας
  • Ημ/νία και ώρα βάρδιας
  • Τρέχουσα κατάσταση (Εκκρεμεί / Εγκεκριμένη / Απορρίφθηκε / Ακυρωμένη)
  • Λόγος απόρριψης (αν υπάρχει)

6.2 Φιλτράρισμα

Χρησιμοποιήστε τα φίλτρα κατάστασης για να δείτε μόνο τις εγκεκριμένες, εκκρεμείς κ.λπ.

7. Οι Πόντοι μου

7.1 Πώς κερδίζω πόντους

Κερδίζετε πόντους κάθε φορά που παρουσιάζεστε σε μια βάρδια. Οι πόντοι υπολογίζονται αυτόματα μετά την καταγραφή παρουσίας από τον διαχειριστή.

7.2 Υπολογισμός πόντων

ΚριτήριοΠόντοι
Βασικοί πόντοι10 πόντοι ανά ώρα
Σαββατοκύριακο×1.5 (50% bonus)
Νυχτερινή βάρδια×1.5 (50% bonus)
Ιατρική / Διασωστική×2.0 (100% bonus)
Παράδειγμα Εθελοντεύσατε 3 ώρες ένα Σάββατο βράδυ:
3 ώρες × 10 πόντοι × 1.5 (Σ/Κ) × 1.5 (νύχτα) = 67.5 → 68 πόντοι

7.3 Ιστορικό πόντων

Στη σελίδα Οι Πόντοι μου δείτε:

  • Συνολικούς πόντους
  • Αναλυτικό ιστορικό: ποιά βάρδια, πόσοι πόντοι, ημ/νία

8. Εργασίες

8.1 Προβολή εργασιών

Στη σελίδα Εργασίες βλέπετε τις εργασίες που σας έχουν ανατεθεί. Κάθε εργασία έχει:

  • Τίτλος και Περιγραφή
  • Προτεραιότητα: Χαμηλή / Κανονική / Υψηλή / Επείγουσα
  • Προθεσμία: Ημ/νία λήξης
  • Κατάσταση: Νέα / Σε εξέλιξη / Ολοκληρωμένη

8.2 Ενημέρωση κατάστασης

  1. Ανοίξτε την εργασία
  2. Αλλάξτε την κατάσταση (π.χ. «Σε εξέλιξη» ή «Ολοκληρωμένη»)
  3. Προσθέστε σχόλιο αν χρειάζεται
  4. Πατήστε Αποθήκευση

9. Εκπαίδευση

9.1 Μαθήματα

Στη σελίδα Μαθήματα βρείτε τα διαθέσιμα εκπαιδευτικά μαθήματα. Κάθε μάθημα έχει:

  • Τίτλο και περιγραφή
  • Κατηγορία (π.χ. Πρώτες Βοήθειες, Ασφάλεια)
  • Διάρκεια

9.2 Εκπαιδευτικό υλικό

Στη σελίδα Εκπαιδευτικό Υλικό μπορείτε να κατεβάσετε αρχεία:

  • PDF εγχειρίδια
  • Παρουσιάσεις
  • Βίντεο (σύνδεσμοι)

Χρησιμοποιήστε τα φίλτρα κατηγορίας για να βρείτε γρήγορα αυτό που ψάχνετε.

Συμβουλή Μελετήστε το εκπαιδευτικό υλικό πριν δώσετε διαγώνισμα — σας βοηθά να πετύχετε υψηλότερο βαθμό!

10. Διαγωνίσματα & Κουίζ

10.1 Κουίζ (Εξάσκηση)

Τα Κουίζ είναι ελαφριές ασκήσεις χωρίς χρονικό περιορισμό. Ιδανικά για εξάσκηση πριν τα διαγωνίσματα.

  1. Μεταβείτε στα Κουίζ
  2. Επιλέξτε ένα κουίζ
  3. Απαντήστε τις ερωτήσεις
  4. Δείτε τα αποτελέσματά σας αμέσως

10.2 Διαγωνίσματα (Εξετάσεις)

Τα Διαγωνίσματα είναι επίσημες εξετάσεις με χρονικό όριο και ελάχιστο βαθμό επιτυχίας.

  1. Μεταβείτε στα Διαγωνίσματα
  2. Δείτε τα διαθέσιμα — κάθε ένα δείχνει: Κατηγορία, Ερωτήσεις, Ποσοστό επιτυχίας, Χρονικό όριο
  3. Πατήστε Έναρξη
  4. Απαντήστε σε όλες τις ερωτήσεις πριν τελειώσει ο χρόνος
  5. Πατήστε Υποβολή
  6. Δείτε τα αποτελέσματά σας: βαθμός, σωστές/λάθος, αν περάσατε
Προσοχή Μόλις ξεκινήσετε ένα διαγώνισμα, ο χρόνος τρέχει! Αν κλείσετε τη σελίδα, ο χρόνος συνεχίζει να μετρά.

10.3 Ιστορικό εξετάσεων

Στο Προφίλ σας και στη σελίδα Αποτελέσματα, βλέπετε όλες τις προσπάθειές σας: βαθμός, αν περάσατε, ημερομηνία.

11. Απόθεμα & Εξοπλισμός

Πρόσβαση Αν είστε Δόκιμος Διασώστης (Trainee Rescuer), δεν έχετε πρόσβαση σε αυτή την ενότητα. Η πρόσβαση ενεργοποιείται μετά την αναβάθμισή σας.

11.1 Υλικά

Στη σελίδα Υλικά δείτε τον διαθέσιμο εξοπλισμό. Κάθε υλικό δείχνει:

  • Όνομα και κατηγορία
  • Διαθεσιμότητα (διαθέσιμο / χρεωμένο)
  • Τοποθεσία αποθήκευσης

11.2 Σετ εξοπλισμού

Τα σετ εξοπλισμού ομαδοποιούν πολλά υλικά (π.χ. Σετ Πρώτων Βοηθειών). Μπορείτε να δείτε τι περιέχει κάθε σετ.

11.3 Χρέωση / Επιστροφή

Η χρέωση (δανεισμός) και επιστροφή εξοπλισμού γίνεται από τον διαχειριστή ή τον αρχηγό βάρδιας. Εσείς μπορείτε να δείτε ποιον εξοπλισμό έχετε χρεωμένο.

11.4 Υλικά ραφιού

Δείτε τα υλικά ανά ράφι αποθήκευσης, μαζί με ημερομηνίες λήξης.

12. Πιστοποιητικά

12.1 Τα πιστοποιητικά μου

Στο Προφίλ σας, στην ενότητα «Τα Πιστοποιητικά μου», βλέπετε όλα τα πιστοποιητικά που σας έχουν εκδοθεί:

  • Τύπος: Πιστοποίηση Πρώτων Βοηθειών, Κολύμβηση κ.λπ.
  • Ημ/νία έκδοσης
  • Ημ/νία λήξης
  • Κατάσταση:
    • Ενεργό — σε ισχύ
    • Λήγει σύντομα — θα λήξει σύντομα
    • Ληγμένο — χρειάζεται ανανέωση
Αυτόματη ειδοποίηση Θα λάβετε email ειδοποίηση πριν τη λήξη πιστοποιητικού, ώστε να προλάβετε την ανανέωση.

13. Κατάταξη & Επιτεύγματα

13.1 Κατάταξη (Leaderboard)

Η σελίδα Κατάταξη εμφανίζει τη βαθμολογία όλων των εθελοντών. Δείτε πού βρίσκεστε!

Φίλτρα:

  • Περίοδος: Αυτόν τον μήνα, Φέτος, Όλων των εποχών
  • Τμήμα: Σύγκριση μόνο με εθελοντές του τμήματός σας

13.2 Επιτεύγματα (Achievements)

Τα Επιτεύγματα είναι βραβεία που κερδίζετε αυτόματα βάσει της δραστηριότητάς σας. Παραδείγματα:

ΕπίτευγμαΠώς το κερδίζετε
Πρώτη ΒάρδιαΟλοκληρώστε την πρώτη σας βάρδια
10 ΑποστολέςΣυμμετέχετε σε 10 αποστολές
100 ΏρεςΣυμπληρώστε 100 ώρες εθελοντισμού
ΚορυφαίοςΦτάστε στην 1η θέση της κατάταξης

Στη σελίδα Επιτεύγματα δείτε ποια έχετε κερδίσει (χρωματισμένα) και ποια σας λείπουν (αχνά). Η μπάρα ποσοστού δείχνει τη συνολική πρόοδό σας.

14. Ειδοποιήσεις

14.1 Τύποι ειδοποιήσεων

Το σύστημα σάς ειδοποιεί για:

  • Έγκριση ή απόρριψη αίτησης συμμετοχής
  • Ακύρωση αποστολής
  • Υπενθύμιση βάρδιας (πριν την έναρξη)
  • Νέα εργασία
  • Προσθήκη σας σε βάρδια από admin
  • Λήξη πιστοποιητικού

14.2 Πού βλέπω τις ειδοποιήσεις

  • Εικονίδιο καμπάνας πάνω δεξιά — αριθμός αδιάβαστων
  • Σελίδα Ειδοποιήσεις — πλήρης λίστα
  • Email — λαμβάνετε και email (εφόσον είναι ενεργοποιημένο)

14.3 Ρυθμίσεις ειδοποιήσεων

Μεταβείτε σε Ρυθμίσεις Ειδοποιήσεων (από το dropdown πάνω δεξιά) για να ελέγξετε ποιες ειδοποιήσεις θέλετε να λαμβάνετε μέσω email.

15. Συχνές Ερωτήσεις

Πώς αλλάζω τα στοιχεία μου;

Μεταβείτε στο Το Προφίλ μου → Ενημερώστε τα πεδία → Αποθήκευση.

Πώς αλλάζω τον κωδικό μου;

Στο Προφίλ, χρησιμοποιήστε την κάρτα «Αλλαγή Κωδικού» (βλ. Ενότητα 3.4).

Μπορώ να ακυρώσω μια εγκεκριμένη συμμετοχή;

Όχι απευθείας. Επικοινωνήστε με τον αρμόδιο διαχειριστή ή αρχηγό βάρδιας.

Γιατί δεν βλέπω το Απόθεμα;

Αν είστε Δόκιμος Διασώστης (Trainee Rescuer), δεν έχετε πρόσβαση. Ζητήστε αναβάθμιση από τον διαχειριστή.

Πώς κερδίζω πόντους;

Παρουσιαστείτε στις βάρδιές σας. Μετά την καταγραφή παρουσίας, οι πόντοι υπολογίζονται αυτόματα (βλ. Ενότητα 7).

Πώς ανεβαίνω στην κατάταξη;

Συμμετέχετε σε αποστολές, ιδίως Σ/Κ, νυχτερινές και διασωστικές — δίνουν περισσότερους πόντους!

Δεν λαμβάνω emails. Τι κάνω;

Ελέγξτε τον φάκελο spam/junk. Αν δεν βρείτε τίποτα, ενημερώστε τον διαχειριστή — μπορεί να υπάρχει πρόβλημα στις ρυθμίσεις SMTP.

Πώς μπορώ να δω τα πιστοποιητικά μου;

Στο Το Προφίλ μου, στην ενότητα «Τα Πιστοποιητικά μου» (βλ. Ενότητα 12).

Τι κάνω αν ξεχάσω τον κωδικό;

Στη σελίδα σύνδεσης, πατήστε «Ξέχασα τον κωδικό» και ακολουθήστε τις οδηγίες (βλ. Ενότητα 1.3).
