1. Σύνδεση & Πίνακας Ελέγχου

1.1 Σύνδεση στο σύστημα

Ανοίξτε τη διεύθυνση της πλατφόρμας στον browser σας. Στη σελίδα σύνδεσης εισάγετε:

  • Email: Η ηλεκτρονική διεύθυνση του λογαριασμού σας
  • Κωδικός: Ο κωδικός πρόσβασης

Πατήστε Σύνδεση για να εισέλθετε. Αν ξεχάσατε τον κωδικό, πατήστε «Ξέχασα τον κωδικό» για να λάβετε email επαναφοράς.

1.2 Πίνακας Ελέγχου (Dashboard)

Μετά τη σύνδεση, ο Πίνακας Ελέγχου εμφανίζει μια γενική εικόνα του συστήματος:

  • Κάρτες στατιστικών: Σύνολο ενεργών αποστολών, εθελοντών, ωρών εθελοντισμού, εκκρεμών αιτήσεων
  • Γραφήματα: Εξέλιξη αποστολών ανά μήνα, κατανομή βαρδιών, στατιστικά συμμετοχών
  • Πρόσφατη δραστηριότητα: Τελευταίες ενέργειες στο σύστημα
Συμβουλή Ο Πίνακας Ελέγχου ανανεώνεται αυτόματα κάθε φορά που τον επισκέπτεστε. Χρησιμοποιήστε τον ως αφετηρία για γρήγορη πρόσβαση σε όλες τις λειτουργίες.

1.3 Πλοήγηση

Η πλαϊνή μπάρα (sidebar) στα αριστερά περιέχει όλες τις ενότητες. Ως διαχειριστής βλέπετε επιπλέον μενού που δεν είναι ορατά στους εθελοντές, όπως:

  • Πρόχειρες, Κλειστές και Ολοκληρωμένες Αποστολές
  • Διαχείριση Εθελοντών
  • Ρυθμίσεις Συστήματος
  • Αρχείο Καταγραφής

2. Διαχείριση Αποστολών

2.1 Κύκλος ζωής αποστολής

Κάθε αποστολή περνάει από τα εξής στάδια:

ΚατάστασηΠεριγραφήΕπόμενο βήμα
Πρόχειρη (DRAFT)Η αποστολή δημιουργήθηκε αλλά δεν είναι ορατή στους εθελοντέςΔημοσίευση → Ανοιχτή
Ανοιχτή (OPEN)Ορατή στους εθελοντές, δέχεται αιτήσεις συμμετοχήςΚλείσιμο → Κλειστή
Κλειστή (CLOSED)Δεν δέχεται νέες αιτήσεις, αναμένει ολοκλήρωσηΟλοκλήρωση ή Ακύρωση
Ολοκληρωμένη (COMPLETED)Η αποστολή ολοκληρώθηκε επιτυχώςΤελικό στάδιο
Ακυρωμένη (CANCELED)Η αποστολή ακυρώθηκεΤελικό στάδιο

2.2 Δημιουργία αποστολής

  1. Μεταβείτε στο μενού Αποστολές και πατήστε + Νέα Αποστολή
  2. Συμπληρώστε τα πεδία:
    • Τίτλος: Σύντομη περιγραφή της αποστολής
    • Περιγραφή: Αναλυτικές πληροφορίες
    • Τμήμα: Ποιο τμήμα είναι υπεύθυνο
    • Τύπος αποστολής: Κατηγορία (π.χ. Εκπαιδευτική, Επιχειρησιακή)
    • Ημ/νία έναρξης & λήξης: Χρονικό πλαίσιο
    • Τοποθεσία: Χώρος διεξαγωγής
  3. Πατήστε Αποθήκευση. Η αποστολή αποθηκεύεται ως Πρόχειρη (DRAFT)
  4. Προσθέστε βάρδιες (βλ. Ενότητα 3) πριν τη δημοσίευση
  5. Πατήστε Δημοσίευση για να γίνει Ανοιχτή και ορατή στους εθελοντές

2.3 Επεξεργασία αποστολής

Κάντε κλικ σε μια αποστολή για να δείτε τις λεπτομέρειές της. Πατήστε Επεξεργασία για να τροποποιήσετε τα στοιχεία. Μπορείτε να επεξεργαστείτε μια αποστολή σε οποιαδήποτε κατάσταση εκτός από Ολοκληρωμένη και Ακυρωμένη.

2.4 Αλλαγή κατάστασης

Στη σελίδα προβολής αποστολής, χρησιμοποιήστε τα κουμπιά αλλαγής κατάστασης:

  • Δημοσίευση: DRAFT → OPEN
  • Κλείσιμο: OPEN → CLOSED
  • Ολοκλήρωση: CLOSED → COMPLETED
  • Ακύρωση: Χρειάζεται αιτιολόγηση. Αποστέλλεται ειδοποίηση σε όλους τους εγκεκριμένους εθελοντές
Προσοχή Η ακύρωση αποστολής είναι μη αναστρέψιμη. Όλοι οι εθελοντές με εγκεκριμένη συμμετοχή θα ειδοποιηθούν μέσω email και ειδοποίησης στο σύστημα.

2.5 Τύποι αποστολών

Διαχειριστείτε τους τύπους αποστολών μέσω Διαχείριση → Τύποι Αποστολών. Κάθε τύπος μπορεί να έχει δικό του εικονίδιο και χρώμα. Παραδείγματα: Διασωστική, Εκπαιδευτική, Κοινοτική, TEP.

2.6 Φιλτράρισμα & αναζήτηση

Στη λίστα αποστολών χρησιμοποιήστε τα φίλτρα για:

  • Κατάσταση: Πρόχειρες, Ανοιχτές, Κλειστές, Ολοκληρωμένες
  • Τμήμα: Φιλτράρετε ανά τμήμα
  • Τύπος: Φιλτράρετε ανά τύπο
  • Αναζήτηση: Ελεύθερο κείμενο στον τίτλο

2.7 Ολοκλήρωση αποστολής (Debrief)

Μετά το κλείσιμο μιας αποστολής, μπορείτε να συμπληρώσετε το Debrief (Απολογισμός) με σημειώσεις, παρατηρήσεις και αξιολόγηση. Αυτό βοηθά στη βελτίωση μελλοντικών αποστολών.

3. Διαχείριση Βαρδιών

3.1 Τι είναι η βάρδια

Κάθε αποστολή αποτελείται από μία ή περισσότερες βάρδιες. Η βάρδια ορίζει το συγκεκριμένο χρονικό παράθυρο και τον μέγιστο αριθμό εθελοντών. Οι εθελοντές αιτούνται συμμετοχή σε βάρδιες, όχι απευθείας σε αποστολές.

3.2 Δημιουργία βάρδιας

  1. Ανοίξτε τη σελίδα της αποστολής
  2. Πατήστε + Προσθήκη Βάρδιας
  3. Συμπληρώστε:
    • Ώρα έναρξης & λήξης: Ωράριο βάρδιας
    • Μέγιστος αριθμός εθελοντών: Πόσοι μπορούν να συμμετέχουν
    • Σημειώσεις: Πρόσθετες πληροφορίες
  4. Πατήστε Αποθήκευση

3.3 Προβολή βάρδιας

Στη σελίδα προβολής βάρδιας βλέπετε:

  • Στοιχεία βάρδιας (ώρες, αποστολή, θέσεις)
  • Λίστα αιτήσεων συμμετοχής (εκκρεμείς, εγκεκριμένες, απορριφθείσες)
  • Κουμπιά ενεργειών (έγκριση, απόρριψη, ακύρωση)

3.4 Διαγραφή βάρδιας

Η διαγραφή βάρδιας εμφανίζει modal επιβεβαίωσης. Αν υπάρχουν εγκεκριμένοι εθελοντές, εμφανίζεται η λίστα τους και ειδοποιούνται αυτόματα.

Σημαντικό Η διαγραφή βάρδιας διαγράφει μόνιμα όλες τις αιτήσεις συμμετοχής. Χρησιμοποιήστε με προσοχή.

4. Συμμετοχές & Παρουσίες

4.1 Ροή αίτησης συμμετοχής

ΚατάστασηΠεριγραφή
ΕκκρεμείΟ εθελοντής υπέβαλε αίτηση — αναμένει απόφαση
ΕγκεκριμένηΗ αίτηση εγκρίθηκε
ΑπορρίφθηκεΗ αίτηση απορρίφθηκε (χρειάζεται αιτιολόγηση)
ΑκυρωμένηΑκυρώθηκε από τον εθελοντή ή τον διαχειριστή

4.2 Έγκριση / Απόρριψη αιτήσεων

  1. Μεταβείτε στη Βάρδια ή στη σελίδα Συμμετοχές
  2. Στην ενότητα εκκρεμών αιτήσεων, πατήστε Έγκριση ή Απόρριψη
  3. Σε περίπτωση απόρριψης, καταχωρήστε τον λόγο στο πεδίο «Λόγος απόρριψης»
  4. Ο εθελοντής ενημερώνεται αυτόματα μέσω email και ειδοποίησης

4.3 Προσθήκη εθελοντή χειροκίνητα

Ως διαχειριστής μπορείτε να προσθέσετε εθελοντή απευθείας σε βάρδια χωρίς αίτηση. Στη σελίδα προβολής βάρδιας, χρησιμοποιήστε το «Προσθήκη Εθελοντή» και επιλέξτε από τη λίστα.

4.4 Επανενεργοποίηση αίτησης

Αν μια αίτηση απορρίφθηκε λανθασμένα, μπορείτε να πατήσετε Επανενεργοποίηση για να την επαναφέρετε σε κατάσταση «Εκκρεμεί».

4.5 Καταχώρηση παρουσίας

  1. Μετά τη λήξη της βάρδιας, ανοίξτε τη σελίδα προβολής βάρδιας
  2. Για κάθε εγκεκριμένο εθελοντή, τσεκάρετε αν Παρευρέθηκε
  3. Εισάγετε τις πραγματικές ώρες αν διαφέρουν από τις προγραμματισμένες
  4. Πατήστε Αποθήκευση Παρουσίας
  5. Οι πόντοι υπολογίζονται αυτόματα βάσει ωρών και πολλαπλασιαστών

4.6 Υπολογισμός πόντων

Οι πόντοι υπολογίζονται ως εξής:

Βασικοί πόντοι10 πόντοι × ώρες
Σαββατοκύριακο×1.5 πολλαπλασιαστής
Νυχτερινή βάρδια×1.5 πολλαπλασιαστής
Ιατρική/Διασωστική×2.0 πολλαπλασιαστής
Παράδειγμα Εθελοντής που δούλεψε 4 ώρες σε νυχτερινή βάρδια Σαββάτου: 4 × 10 × 1.5 (Σ/Κ) × 1.5 (νύχτα) = 90 πόντοι.

5. Εργασίες

5.1 Σύστημα εργασιών

Οι εργασίες (tasks) επιτρέπουν στους διαχειριστές να αναθέτουν μεμονωμένες δραστηριότητες σε εθελοντές εκτός αποστολών.

5.2 Δημιουργία εργασίας

  1. Μεταβείτε στο Εργασίες → + Νέα Εργασία
  2. Συμπληρώστε τίτλο, περιγραφή, προτεραιότητα, ημερομηνία λήξης
  3. Αναθέστε σε έναν ή περισσότερους εθελοντές
  4. Πατήστε Αποθήκευση

5.3 Παρακολούθηση

Ως διαχειριστής βλέπετε όλες τις εργασίες. Χρησιμοποιήστε τα φίλτρα κατάστασης (Νέα, Σε εξέλιξη, Ολοκληρωμένη) και ανάθεσης για εύκολη παρακολούθηση.

6. Εθελοντές & Χρήστες

6.1 Κατάλογος εθελοντών

Στο Διοίκηση → Εθελοντές εμφανίζεται πλήρης λίστα εθελοντών με φίλτρα ρόλου, τμήματος, τύπου εθελοντή και αναζήτησης.

6.2 Ρόλοι χρηστών

ΡόλοςΔικαιώματα
Διαχειριστής ΣυστήματοςΠλήρη πρόσβαση σε όλες τις λειτουργίες, ρυθμίσεις, αρχεία
Διαχειριστής ΤμήματοςΔιαχείριση αποστολών, εθελοντών, πιστοποιητικών του τμήματός τους
Αρχηγός ΒάρδιαςΔιαχείριση βαρδιών, παρουσιών, πρόσβαση σε επιχειρησιακό πίνακα
ΕθελοντήςΑίτηση συμμετοχής, εκπαίδευση, προφίλ

6.3 Τύποι εθελοντών

  • Εθελοντής (Volunteer): Πλήρης πρόσβαση σε απόθεμα και λειτουργίες
  • Διασώστης (Rescuer): Πρόσβαση σε εξοπλισμό, εκπαιδευτικές αποστολές, εξετάσεις
  • Δόκιμος Διασώστης (Trainee Rescuer): Περιορισμένη πρόσβαση — αποκλεισμός από απόθεμα

6.4 Δημιουργία / Επεξεργασία εθελοντή

  1. Πατήστε + Νέος Εθελοντής ή επιλέξτε υπάρχοντα
  2. Συμπληρώστε: Όνομα, Email, Τηλέφωνο, Ρόλος, Τύπος εθελοντή, Τμήμα, Παράρτημα
  3. Πατήστε Αποθήκευση

6.5 Μαζική εισαγωγή (CSV Import)

Μέσω Εθελοντές → Εισαγωγή CSV μπορείτε να εισάγετε πολλούς εθελοντές ταυτόχρονα:

  1. Βήμα 1: Ανεβάστε το αρχείο CSV
  2. Βήμα 2: Ελέγξτε την προεπισκόπηση — το σύστημα εμφανίζει τυχόν σφάλματα
  3. Βήμα 3: Επιβεβαιώστε την εισαγωγή

Το CSV πρέπει να περιέχει στήλες: Όνομα, Email, Τηλέφωνο (τουλάχιστον).

6.6 Ανενεργοί εθελοντές

Εθελοντές που δεν συμμετέχουν ενεργά μπορούν να χαρακτηριστούν ως ανενεργοί. Εμφανίζονται σε ξεχωριστή λίστα (Ανενεργοί Εθελοντές) και δεν δέχονται αιτήσεις.

6.7 Αναφορά εθελοντή

Η σελίδα Αναφορά Εθελοντή εμφανίζει αναλυτικά στοιχεία: ιστορικό συμμετοχών, πόντους, παρουσίες, πιστοποιητικά, εξετάσεις.

6.8 Θέσεις εθελοντών

Διαχειριστείτε τις θέσεις εθελοντών (π.χ. Υπεύθυνος Αποθήκης, Οδηγός) μέσω Θέσεις Εθελοντών.

7. Εκπαίδευση & Εξετάσεις

7.1 Κατηγορίες εκπαίδευσης

Η εκπαίδευση χωρίζεται σε κατηγορίες (π.χ. Πρώτες Βοήθειες, Ασφάλεια, Επικοινωνία). Διαχειριστείτε τις κατηγορίες μέσω Διαχ. Εκπαίδευσης → Κατηγορίες & Υλικό.

7.2 Εκπαιδευτικό υλικό

Ανεβάστε αρχεία (PDF, παρουσιάσεις, βίντεο links) ως εκπαιδευτικό υλικό σε κάθε κατηγορία. Οι εθελοντές βλέπουν το υλικό μέσα από τη σελίδα «Εκπαιδευτικό Υλικό».

7.3 Μαθήματα

Δημιουργήστε μαθήματα με περιγραφή, κατηγορία, διάρκεια. Τα μαθήματα εμφανίζονται στη σελίδα «Μαθήματα» για τους εθελοντές.

7.4 Διαγωνίσματα (Exams)

  1. Μεταβείτε σε Διαχ. Εκπαίδευσης → Νέο Διαγώνισμα
  2. Ορίστε: Τίτλος, Κατηγορία, Ποσοστό επιτυχίας, Ερωτήσεις ανά προσπάθεια, Χρονικό όριο
  3. Προσθέστε ερωτήσεις από τη Δεξαμενή Ερωτήσεων (questions pool)
  4. Δημοσιεύστε το διαγώνισμα
Δεξαμενή ερωτήσεων Η Δεξαμενή Ερωτήσεων είναι κοινή βιβλιοθήκη ερωτήσεων. Μπορείτε να δημιουργήσετε ερωτήσεις εκεί και να τις χρησιμοποιήσετε σε πολλά διαγωνίσματα.

7.5 Κουίζ

Τα κουίζ λειτουργούν παρόμοια με τα διαγωνίσματα αλλά είναι ελαφρύτερα — χωρίς χρονικό όριο, για εξάσκηση.

7.6 Στατιστικά εξετάσεων

Στη σελίδα Στατιστικά Εξετάσεων δείτε: μέσος βαθμός, ποσοστό επιτυχίας, κατανομή βαθμών, σύγκριση ανά κατηγορία. Εξαγωγή δεδομένων σε CSV/Excel.

8. Απόθεμα & Εξοπλισμός

8.1 Γενική εικόνα

Το σύστημα αποθέματος διαχειρίζεται τον εξοπλισμό του οργανισμού: ποιος τον κατέχει, πού βρίσκεται, πότε λήγει.

8.2 Υλικά (Inventory Items)

  1. Μεταβείτε σε Απόθεμα → Υλικά
  2. Πατήστε + Νέο Υλικό για δημιουργία
  3. Συμπληρώστε: Όνομα, Κατηγορία, Αριθμός αποθέματος, Τοποθεσία, Ημ/νία λήξης κ.λπ.
  4. Πατήστε Αποθήκευση

8.3 Σετ εξοπλισμού (Kits)

Ομαδοποιήστε πολλά υλικά σε Σετ (π.χ. Σετ Πρώτων Βοηθειών). Κατά τη χρέωση ενός σετ, χρεώνονται αυτόματα όλα τα υλικά του.

8.4 Χρέωση / Επιστροφή

Μέσω Χρέωση / Επιστροφή:

  • Επιλέξτε εθελοντή και υλικό/σετ
  • Πατήστε Χρέωση για ανάθεση
  • Πατήστε Επιστροφή για παραλαβή

8.5 Αποθήκες & Ράφια

Οργανώστε τα υλικά σε Αποθήκες και Ράφια. Κάθε ράφι μπορεί να έχει ημερομηνία λήξης — το σύστημα στέλνει υπενθύμιση πριν τη λήξη.

8.6 Κατηγορίες & Τοποθεσίες

Διαχειριστείτε κατηγορίες υλικών (π.χ. Ιατρικός εξοπλισμός, Εργαλεία) και τοποθεσίες αποθήκευσης.

8.7 Σημειώσεις υλικών

Προσθέστε σημειώσεις σε κάθε υλικό (π.χ. «Χρειάζεται σέρβις», «Λείπει εξάρτημα»).

8.8 Ετικέτες (Labels)

Εκτυπώστε ετικέτες barcode/QR για τα υλικά μέσω Εκτύπωση ετικετών.

Πρόσβαση Οι Δόκιμοι Διασώστες (Trainee Rescuer) δεν έχουν πρόσβαση στο σύστημα αποθέματος. Πρέπει πρώτα να αναβαθμιστούν σε Εθελοντή ή Διασώστη.

9. Πιστοποιητικά

9.1 Τύποι πιστοποιητικών

Διαχειριστείτε τους τύπους πιστοποιητικών μέσω Πιστοποιητικά → Τύποι (μόνο Διαχειριστής Συστήματος). Κάθε τύπος ορίζει: Όνομα, Διάρκεια ισχύος, Απαιτήσεις.

9.2 Πιστοποιητικά εθελοντών

Στη σελίδα Πιστοποιητικά βλέπετε τα πιστοποιητικά όλων των εθελοντών. Φίλτρα ανά τύπο, κατάσταση (ενεργό, λήξη σύντομα, ληγμένο), εθελοντή.

9.3 Έκδοση πιστοποιητικού

  1. Ανοίξτε το προφίλ εθελοντή ή τη σελίδα Πιστοποιητικά
  2. Πατήστε + Νέο Πιστοποιητικό
  3. Επιλέξτε τύπο, εθελοντή, ημ/νία έκδοσης, ημ/νία λήξης
  4. Πατήστε Αποθήκευση

9.4 Ειδοποιήσεις λήξης

Το σύστημα αποστέλλει αυτόματα ειδοποιήσεις πριν τη λήξη πιστοποιητικών (μέσω cron job). Η ρύθμιση του χρόνου ειδοποίησης γίνεται στις Ρυθμίσεις Συστήματος.

9.5 Εξαγωγή

Εξάγετε λίστα πιστοποιητικών σε CSV για reporting.

10. Πολίτες

Πρόσβαση Η ενότητα Πολιτών είναι διαθέσιμη μόνο σε Διαχειριστές Συστήματος και Διαχειριστές Τμήματος.

10.1 Κατάλογος πολιτών

Διαχειριστείτε πολίτες που εξυπηρετεί ο οργανισμός. Κάθε πολίτης έχει: Ελληνικό όνομα, Λατινικό όνομα, Στοιχεία επικοινωνίας, Κατάσταση.

10.2 Πιστοποιητικά πολιτών

Εκδίδετε πιστοποιητικά για πολίτες (π.χ. Πιστοποίηση Πρώτων Βοηθειών) παρόμοια με τα πιστοποιητικά εθελοντών.

10.3 Τύποι πιστοποιητικών πολιτών

Οι τύποι πιστοποιητικών πολιτών διαχειρίζονται ξεχωριστά από αυτούς των εθελοντών.

11. Ενημερωτικά Δελτία (Newsletters)

Πρόσβαση Μόνο ο Διαχειριστής Συστήματος μπορεί να δημιουργήσει και να αποστείλει ενημερωτικά.

11.1 Δημιουργία newsletter

  1. Μεταβείτε σε Επικοινωνία → Ενημερωτικά
  2. Πατήστε + Νέο Ενημερωτικό
  3. Συμπληρώστε τίτλο, περιεχόμενο (HTML editor), παραλήπτες
  4. Αποθηκεύστε ως Πρόχειρο ή αποστείλετε άμεσα

11.2 Αποστολή

Μετά τη δημιουργία, πατήστε Αποστολή. Το newsletter αποστέλλεται σε όλους τους επιλεγμένους παραλήπτες.

11.3 Ιστορικό αποστολών

Στη σελίδα Newsletter Log δείτε ιστορικό αποστολών: ποιος παρέλαβε, αποτυχίες, στατιστικά.

11.4 Απεγγραφή

Κάθε newsletter περιλαμβάνει σύνδεσμο απεγγραφής. Οι εθελοντές μπορούν να απεγγραφούν χωρίς σύνδεση.

12. Οργανωτική Δομή

12.1 Παραρτήματα

Τα Παραρτήματα (Branches) είναι τα γεωγραφικά/οργανωτικά τμήματα του οργανισμού. Διαχειρίζονται μόνο από τον Διαχειριστή Συστήματος. Κάθε παράρτημα μπορεί να έχει πολλά τμήματα.

12.2 Τμήματα

Τα Τμήματα (Departments) εντάσσονται σε παραρτήματα. Κάθε εθελοντής ανήκει σε ένα τμήμα. Κάθε αποστολή σχετίζεται με ένα τμήμα.

12.3 Δεξιότητες

Διαχειριστείτε τις Δεξιότητες (Skills) που μπορούν να καταχωρηθούν στο προφίλ των εθελοντών. Ομαδοποιούνται σε κατηγορίες (π.χ. Ιατρικές, Τεχνικές, Γλώσσες).

13. Gamification — Πόντοι & Επιτεύγματα

13.1 Σύστημα πόντων

Οι πόντοι αποδίδονται αυτόματα μετά την καταχώρηση παρουσίας. Ο υπολογισμός βασίζεται στις ώρες και τους πολλαπλασιαστές (βλ. Ενότητα 4.6).

13.2 Κατάταξη (Leaderboard)

Η σελίδα Κατάταξη εμφανίζει τη βαθμολογία εθελοντών με φίλτρα:

  • Περίοδος: Μηνιαία, Ετήσια, Όλων των εποχών
  • Τμήμα: Φιλτράρισμα ανά τμήμα

13.3 Επιτεύγματα (Achievements)

Τα επιτεύγματα αποδίδονται αυτόματα βάσει κριτηρίων (π.χ. «10 αποστολές», «100 ώρες», «Πρώτη βάρδια»). Κάθε επίτευγμα έχει εικονίδιο, περιγραφή και κατηγορία.

13.4 Ρύθμιση πολλαπλασιαστών

Ως Διαχειριστής Συστήματος μπορείτε να αλλάξετε τους πολλαπλασιαστές πόντων μέσω Ρυθμίσεις → Gamification:

  • Βασικοί πόντοι ανά ώρα (προεπιλογή: 10)
  • Πολλαπλασιαστής Σ/Κ (προεπιλογή: ×1.5)
  • Πολλαπλασιαστής νύχτας (προεπιλογή: ×1.5)
  • Πολλαπλασιαστής ιατρικού (προεπιλογή: ×2.0)

14. Αναφορές & Εξαγωγές

14.1 Αναφορές

Η σελίδα Αναφορές παρέχει συγκεντρωτικά στατιστικά στοιχεία:

  • Αποστολές ανά κατάσταση, τμήμα, μήνα
  • Ώρες εθελοντισμού ανά εθελοντή
  • Στατιστικά συμμετοχών
  • Αναφορά εθελοντή (αναλυτική)

14.2 Εξαγωγή δεδομένων

Πολλές σελίδες υποστηρίζουν εξαγωγή σε CSV ή Excel:

  • Λίστα εθελοντών
  • Πιστοποιητικά
  • Στατιστικά εξετάσεων
  • Απόθεμα

Πατήστε το κουμπί Εξαγωγή (εικονίδιο ) που εμφανίζεται στις αντίστοιχες σελίδες.

15. Ρυθμίσεις Συστήματος

Μόνο Διαχειριστής Συστήματος Η σελίδα Ρυθμίσεις είναι προσβάσιμη μόνο με ρόλο Διαχειριστή Συστήματος.

15.1 Γενικές ρυθμίσεις

  • Όνομα εφαρμογής: Τίτλος που εμφανίζεται παντού
  • Gamification: Πολλαπλασιαστές πόντων
  • Εγγραφή: Ενεργοποίηση/απενεργοποίηση αυτο-εγγραφής εθελοντών

15.2 Ρυθμίσεις SMTP

Ρυθμίστε τον SMTP server για αποστολή emails:

  • SMTP Host, Port, Username, Password
  • Encryption (TLS/SSL)
  • Αποστολέας (From Name, From Email)

15.3 Email Templates

Οι ειδοποιήσεις αποστέλλονται με βάση πρότυπα email αποθηκευμένα στη βάση δεδομένων. Κάθε πρότυπο έχει:

  • Κωδικός: Μοναδικό αναγνωριστικό (π.χ. participation_approved)
  • Θέμα: Θέμα email
  • Σώμα HTML: Περιεχόμενο με placeholders (π.χ. {{user_name}}, {{mission_title}})

Επεξεργαστείτε τα πρότυπα με τον ενσωματωμένο HTML editor. Πατήστε Προεπισκόπηση για να δείτε πώς θα φαίνεται.

15.4 Ρυθμίσεις ειδοποιήσεων

Ελέγξτε ποιες ειδοποιήσεις στέλνουν email:

  • Έγκριση/Απόρριψη συμμετοχής
  • Ακύρωση αποστολής
  • Υπενθύμιση βάρδιας
  • Προσθήκη εθελοντή από admin

15.5 Ρυθμίσεις αποθέματος

Ρυθμίστε παραμέτρους αποθέματος: ημέρες ειδοποίησης λήξης ραφιού, ενεργοποίηση/απενεργοποίηση modules.

15.6 Ρυθμίσεις πιστοποιητικών πολιτών

Ρυθμίστε τις ημέρες ειδοποίησης πριν τη λήξη πιστοποιητικών πολιτών.

16. Αρχείο Καταγραφής & Email Logs

16.1 Αρχείο καταγραφής (Audit Log)

Στο Σύστημα → Αρχείο Καταγραφής καταγράφονται όλες οι ενέργειες διαχείρισης:

  • Ποιος έκανε τι, πότε
  • Ποιος πίνακας και εγγραφή επηρεάστηκε
  • Λεπτομέρειες αλλαγής

Χρησιμοποιήστε τα φίλτρα (χρήστης, ενέργεια, πίνακας, ημερομηνία) για αναζήτηση.

16.2 Email Logs

Στα Email Logs δείτε ιστορικό αποστολής email: παραλήπτης, θέμα, κατάσταση (επιτυχία/αποτυχία), ημερομηνία.

17. Επιχειρησιακός Πίνακας (Ops Dashboard)

17.1 Γενική εικόνα

Ο Επιχειρησιακός Πίνακας είναι ειδική σελίδα για διαχείριση σε πραγματικό χρόνο κατά τη διάρκεια αποστολών. Διαθέσιμος σε Διαχειριστές και Αρχηγούς Βάρδιας.

17.2 Λειτουργίες

  • Ζωντανή εικόνα: Τρέχουσες βάρδιες, αριθμός ενεργών εθελοντών
  • Γρήγορες ενέργειες: Επικοινωνία με εθελοντές, αλλαγή κατάστασης
  • Χάρτης: Θέσεις εθελοντών (αν το ping location είναι ενεργό)

Β. Συχνές ερωτήσεις

Πώς αλλάζω τον ρόλο ενός εθελοντή;

Μεταβείτε στο Εθελοντές → Επεξεργασία του εθελοντή → αλλάξτε το πεδίο «Ρόλος» → Αποθήκευση.

Πώς ξαναστέλνω ένα email ειδοποίησης;

Αυτή τη στιγμή δεν υπάρχει κουμπί «Αποστολή ξανά». Μπορείτε να αλλάξετε την κατάσταση συμμετοχής (π.χ. Επανενεργοποίηση → Έγκριση) για αυτόματη αποστολή νέου email.

Πώς δημιουργώ αντίγραφο ασφαλείας;

Χρησιμοποιήστε τα εργαλεία MySQL (mysqldump) ή τα αντίγραφα ασφαλείας μέσω phpMyAdmin. Φάκελος αντιγράφων: backups/.

Πώς τρέχω τα cron jobs;

Τα cron jobs (υπενθυμίσεις, λήξεις πιστοποιητικών) πρέπει να προγραμματιστούν στον Task Scheduler (Windows) ή crontab (Linux). Εκτελούνται μέσω CLI:

  • php cron_shift_reminders.php — Υπενθυμίσεις βαρδιών
  • php cron_certificate_expiry.php — Ειδοποιήσεις λήξης πιστοποιητικών
  • php cron_daily.php — Καθημερινές εργασίες
  • php cron_task_reminders.php — Υπενθυμίσεις εργασιών
