Νέα Δυνατότητα — Εφαρμογή στο Κινητό!

VolunteerOps · Απρίλιος 2026

Αγαπητοί εθελοντές,

Έχουμε νέα που θα σας κάνουν χαρούμενους! Το σύστημα VolunteerOps μπορεί πλέον να εγκατασταθεί απευθείας στο κινητό ή tablet σας — σαν κανονική εφαρμογή, χωρίς να χρειάζεται να ανοίγετε τον browser κάθε φορά.

✅ Τι αλλάζει για εσάς:

  • Εικονίδιο στην αρχική οθόνη — ανοίγετε το VolunteerOps με ένα tap, χωρίς Chrome ή Safari
  • Push Ειδοποιήσεις — λαμβάνετε άμεσα ειδοποίηση για νέες αποστολές, αποτελέσματα αιτήσεων, υπενθυμίσεις βάρδιας κ.ά.
  • Γρήγορη πρόσβαση — ακόμα και χωρίς σύνδεση internet βλέπετε τα δεδομένα σας

🔔 Ενεργοποίηση Push Ειδοποιήσεων μέσα από την εφαρμογή

Αφού εγκαταστήσετε την εφαρμογή, ακολουθήστε τα παρακάτω βήματα για να ενεργοποιήσετε τις ειδοποιήσεις:

1
 Συνδεθείτε στο VolunteerOps και πηγαίνετε στο μενού → Ρυθμίσεις Ειδοποιήσεων (ή κλικ στο όνομά σας πάνω δεξιά)
2
 Στην ενότητα «Push Ειδοποιήσεις» πατήστε το κουμπί «Ενεργοποίηση Push»
3
 Αποδεχτείτε την άδεια ειδοποιήσεων που θα εμφανιστεί από το κινητό σας
✓
 Επιλέξτε ποιες ειδοποιήσεις θέλετε (νέες αποστολές, αποτελέσματα, υπενθυμίσεις κ.λπ.) και αποθηκεύστε

🤖 Android — Chrome

Εγκατάσταση εφαρμογής:

  1. Ανοίξτε το Chrome και πηγαίνετε στο site του VolunteerOps
  2. Πατήστε τις 3 τελείες (⋮) πάνω δεξιά
  3. Επιλέξτε «Προσθήκη στην αρχική οθόνη»
  4. Πατήστε «Προσθήκη» στο αναδυόμενο παράθυρο
  5. Το εικονίδιο εμφανίζεται αυτόματα στην αρχική σας οθόνη! 🎉

Ειδοποιήσεις στο Chrome (Android):

  1. Πηγαίνετε στις Ρυθμίσεις του κινητού → Εφαρμογές → Chrome
  2. Επιλέξτε «Ειδοποιήσεις» και ελέγξτε ότι είναι ενεργοποιημένες
  3. Εναλλακτικά: στο Chrome, αγγίξτε τη «κλειδαριά» (🔒) δίπλα στο URL → Ειδοποιήσεις → Αποδοχή

🍎 iPhone / iPad — Safari

Εγκατάσταση εφαρμογής (απαιτείται iOS 16.4+):

  1. Ανοίξτε αποκλειστικά με Safari (δεν λειτουργεί με Chrome στο iPhone)
  2. Πατήστε το εικονίδιο «Κοινοποίηση» (το τετράγωνο με το βελάκι ↑) στο κάτω μέρος
  3. Σκρολάρετε και επιλέξτε «Προσθήκη στην Αρχική Οθόνη»
  4. Πατήστε «Προσθήκη» πάνω δεξιά
  5. Το εικονίδιο του VolunteerOps εμφανίζεται στην αρχική σας! 🎉

Ενεργοποίηση ειδοποιήσεων (iOS 16.4+):

  1. Ανοίξτε την εφαρμογή από το εικονίδιο (όχι από Safari)
  2. Πηγαίνετε στις Ρυθμίσεις Ειδοποιήσεων του VolunteerOps
  3. Πατήστε «Ενεργοποίηση Push» — θα σας ζητηθεί άδεια
  4. Πατήστε «Αποδοχή» στο αναδυόμενο παράθυρο iOS

⚠️ Σημαντικό για iPhone: Οι push ειδοποιήσεις στο iOS λειτουργούν μόνο εφόσον έχετε εγκαταστήσει την εφαρμογή στην αρχική οθόνη μέσω Safari. Αν ανοίγετε από τον browser, οι ειδοποιήσεις δεν θα εμφανίζονται.
